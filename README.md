## Основные команды для запуска приложения:
### Установка зависимостей
    $ npm i

### Запуск приложения на localhost:3000
    $ npm start

### Тестирование приложения с помощью Cypress
    $ npm run cypress

### Тестирование с помощью Unit-тестов
    $ npm run test
